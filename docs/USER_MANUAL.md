# 📖 Night Hunger: Vampire Evo — Руководство игрока

## 🎯 О игре

**Night Hunger: Vampire Evo** — это Telegram Mini App игра в жанре RPG, где вы играете за вампира, который охотится на врагов, прокачивает характеристики и сражается с другими игроками.

---

## 🚀 Быстрый старт

### 1. Запуск игры

1. Откройте Telegram
2. Найдите бота **@NightHungerBot**
3. Нажмите `/start` или кнопку **Play**
4. Разрешите доступ к вашему профилю Telegram

### 2. Первый вход

При первом входе:
- Вам автоматически присваивается уникальный реферальный код
- Вы получаете стартовые характеристики:
  - 🩸 100 HP (здоровье)
  - ⚔️ 10 Strength (сила)
  - 🏃 10 Agility (ловкость)
  - ✨ 0 XP (опыт)
  - 🩸 0 Blood (валюта)

---

## 🎮 Основной геймплей

### 🩸 Охота

**Охота** — основной способ получения опыта и крови.

1. Нажмите кнопку **🩸 Охота** на главном экране
2. Вы встретите одного из врагов:
   - 🐺 **Волк** — слабый враг, хорош для новичков
   - 🗡️ **Охотник на вампиров** — средний враг
   - 🧙‍♀️ **Ведьма** — опасный магический враг
   - 👹 **Демон** — сильный враг для высоких уровней
   - 👼 **Ангел** — самый опасный противник

3. Выберите действие:
   - **⚔️ Атака** — нанесите урон врагу (враг атакует в ответ)
   - **🏃 Побег** — попытайтесь сбежать (шанс зависит от ловкости)
   - **🩸 Поглощение** — восстановите HP после победы

4. После победы вы получаете:
   - ✨ XP для повышения уровня
   - 🩸 Кровь для покупок в магазине

**Важно:** Между охотами есть задержка 5 секунд (anti-cheat).

### ⚡ Прокачка характеристик

Накопив достаточно крови, вы можете улучшить характеристики:

1. Нажмите **⚡ Прокачка**
2. Выберите характеристику:
   - **Сила** — увеличивает урон в бою
   - **Ловкость** — увеличивает шанс побега
   - **HP** — увеличивает максимальное здоровье

3. Стоимость улучшения: `100 × Уровень`

**Совет:** Равномерно прокачивайте все характеристики для баланса.

### 🎒 Инвентарь

В инвентаре хранятся ваши предметы:

- **🧪 Зелья** — восстанавливают HP
- **👕 Скины** — меняют внешний вид
- - **⚡ Баффы** — временные улучшения

#### Использование предметов

1. Откройте **🎒 Инвентарь**
2. Нажмите на предмет
3. Выберите **Использовать**

#### Продажа предметов

1. Откройте **🎒 Инвентарь**
2. Нажмите на предмет
3. Выберите **Продать**
4. Вы получите кровь

### 🏪 Магазин

В магазине можно купить:

- **Зелья лечения** — восстановление HP
- **Скины** — косметические улучшения
- **Баффы** — временные бонусы

**Валюта:**
- 🩸 **Кровь** — зарабатывается на охоте
- ⭐ **Telegram Stars** — покупается за реальные деньги

## 📊 Советы и стратегии

### Для новичков

1. **Начните с волков** — они слабее других врагов
2. **Прокачивайте HP** — больше здоровья = больше выживаемость
3. **Используйте зелья** — не умирайте зря
4. **Пригласите друзей** — быстрый старт с бонусами

### Для опытных игроков

1. **Фармите демонов** — лучший баланс XP/риск
2. **Копите кровь** — для важных улучшений
3. **Следите за метой** — балансируйте характеристики
4. **Участвуйте в ивентах** — эксклюзивные награды

### Боевая тактика

- Если HP < 30% — используйте зелье или сбегите
- Против слабых врагов — всегда атакуйте
- Против сильных — копите силу перед боем

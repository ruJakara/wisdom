# Done Criteria

Задача считается завершённой, когда выполнены все пункты ниже.

## 1) Код/контент работает по назначению

- Обновлённые инструкции дают ожидаемую последовательность работы.
- Формат выхода skill-файлов остаётся однозначным.

## 2) Нет очевидной поломки существующей логики

- Не нарушен конвейер `анализ -> пост`.
- Не удалены критичные ограничения по качеству и этике.

## 3) Соблюдены правила проекта

- Проверены требования из `docs/ai/rules.md`.
- Нет конфликтов между `skills/` и `docs/ai/`.

## 4) Обновлены нужные docs

- При изменении структуры обновлён `project-map.md`.
- При изменении ролей/связей обновлён `architecture.md`.
- При изменении процесса обновлён `workflow.md`.

## 5) Обновлены связанные файлы

- Нет "одиноких" правок только в одном файле, если изменение затрагивает несколько модулей.

## 6) Есть минимальная проверка результата

- Выполнен `task check` (если Task установлен).
- Проведена ручная проверка согласованности текста и форматов.

# New Feature Workflow

## Цель
Добавить новую функциональность без архитектурного хаоса.

## Шаги
1. Зафиксировать цель, входы, выходы и критерии готовности.
2. Прочитать ключевые части проекта и выбрать точки интеграции.
3. Сформировать короткий план реализации.
4. Внести изменения малыми итерациями.
5. Проверить основной и граничные сценарии.
6. Обновить документацию и контракты.
7. Подготовить отчёт по факту.

## Гейт перед завершением
- Поведение соответствует задаче.
- Нет регрессий в смежных сценариях.
- Документация актуальна.

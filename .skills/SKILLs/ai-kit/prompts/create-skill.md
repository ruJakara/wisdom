# Prompt: Create Skill

Сгенерируй новый skill-файл по шаблону `ai-kit/templates/skill-template.md`.

Вход:
- Название skill:
- Для какого типа проекта:
- Какая роль и зона ответственности:
- Какие входы доступны:
- Какие выходы нужны:
- Какие ограничения и запреты:

Требования к результату:
- Формат строго по секциям шаблона.
- Текст прикладной, без общих фраз.
- В `Typical tasks` дай 3-7 реальных задач.
- В `Quality checklist` дай проверяемые пункты.
- Не выдумывай интеграции, которых не указано во входе.

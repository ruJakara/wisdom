# Prompt: Create Rules

Сформируй проектные правила в нормативном стиле.

Вход:
- Тип проекта и стек:
- Какие зоны чаще ломаются:
- Какие практики обязательны:
- Какие практики запрещены:

Требования к результату:
- Разделы: `Обязательно`, `Запрещено`, `Желательно`.
- Формулировки короткие и однозначные.
- Правила должны быть проверяемыми.
- Учитывай скорость AI-assisted разработки и риск регрессий.
- Избегай абстрактных лозунгов.

# Prompt: Create Workflow

Создай workflow для повторяемого процесса в проекте.

Вход:
- Название процесса:
- Контекст проекта:
- Какие риски критичны:
- Какие проверки обязательны:

Требования к результату:
- Формат по `ai-kit/templates/workflow-template.md`.
- 5-9 шагов, каждый с конкретным действием.
- Добавь секцию `Common Failure Modes` с 3-5 типичными ошибками.
- Укажи `Exit Criteria`, которые можно проверить.
- Не добавляй шаги без практической ценности.

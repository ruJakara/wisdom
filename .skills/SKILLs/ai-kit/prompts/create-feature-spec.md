# Prompt: Create Feature Spec

Сгенерируй feature spec по задаче в формате `ai-kit/specs/feature-spec-template.md`.

Вход:
- Тип проекта (игра/бот/автоматизация/контент-система):
- Краткое описание фичи:
- Кто actor/user:
- Какие интеграции затрагиваются:
- Ограничения (срок/стек/совместимость):

Требования:
- Заполни все секции шаблона: Goal, User/Actor, Context, Trigger, Inputs, Outputs, Main flow, Rules, Edge cases, Failure cases, Dependencies, Acceptance criteria, Out of scope.
- Пиши коротко и операционно: без бюрократии и общих фраз.
- Явно отделяй факты от предположений.
- В `Acceptance criteria` дай только проверяемые пункты.
- В `Out of scope` зафиксируй, что сознательно не делается в этой итерации.

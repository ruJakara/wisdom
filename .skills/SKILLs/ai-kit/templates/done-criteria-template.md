# Done Criteria Template

## Task

## Functional Criteria
- [ ] Основной сценарий работает.
- [ ] Граничные случаи обработаны.

## Regression Criteria
- [ ] Существующие критичные потоки не сломаны.
- [ ] Публичные контракты не нарушены (или изменение задокументировано).

## Quality Criteria
- [ ] Изменения локальны и понятны.
- [ ] Нет скрытых временных фиксов.

## Validation Criteria
- [ ] Выполнены необходимые тесты/проверки.
- [ ] Результат воспроизводим.

## Documentation Criteria
- [ ] Обновлены связанные docs.
- [ ] Ограничения и известные риски зафиксированы.

## Final Check
- [ ] Задача готова к передаче/релизу.

# Guidelines для разработки Night Hunger

## Стиль кода
- Язык: TypeScript (Strict mode).
- Фреймворк: React + Vite.
- Стилизация: Tailwind CSS.
- Именование: CamelCase для переменных, PascalCase для компонентов.
- Комментарии: Только для сложной логики, код должен быть самодокументируемым.

## Специфика Telegram
- Всегда валидировать `initData` на бэкенде.
- Не хранить чувствительные данные в localStorage.
- Использовать `tg.expand()` для разворота на весь экран.
- Учитывать темную тему Telegram (`var(--tg-theme-bg-color)`).

## Запреты
- Не использовать `any` в TypeScript.
- Не делать прямые запросы к БД из фронтенда.
- Не хардкодить секретные ключи.

# UI Компоненты и цвета

## Цветовая палитра
- Background: `#1a1a1a` (или `var(--tg-theme-bg-color)`)
- Primary: `#ff0000` (Кроваво-красный)
- Secondary: `#800080` (Фиолетовый)
- Text: `#ffffff`

## Компоненты
- `Button`: Высота 48px, скругление 12px, активное состояние - затемнение.
- `Card`: Фон `#2a2a2a`, тень, отступ 16px.
- `ProgressBar`: Зеленый (HP), Синий (XP).
